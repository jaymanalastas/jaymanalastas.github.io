def text_process(txt):
    """"
    Removes the following: 
            URLs
            Punctuations
            Stop words
    Return list of clean text words
    
    Parameters
    -------------------------------------------------------------------
    txt: string 
        A text string that you want to parse 
        
    Returns
    -------------------------------------------------------------------
    Clean words 
        
    """
    nonpunc = " ".join(re.sub("([^0-9A-Za-z \t])|(\w+:\/\/\S+)", "", txt).split())
    #nonpunc = [char for char in txt if char not in string.punctuation]
    nonpunc = ''.join(nonpunc)
    return [word for word in nonpunc.split() if word.lower() not in stopwords.words('english')]

Exploring Co-occuring Words (Bigrams)¶

To identify co-occurrence of words in the tweets, we'll use bigrams form nltk.

Let's create a list comprehension to create a list of all bigrams (i.e. co-occuring words) in the tweets.

In [6]:
bigram_tweets = [list(bigrams(tweet)) for tweet in clean_twits]

Visualize Networks of Bigrams¶

We'll use the Data Frame to visualize the top 20 occuring bigrams as networks using the pythong package NetworkX.

In [10]:
# Create a dictionary of bigrams and their counts
d = df.set_index('bigram').T.to_dict('records')

# Create a network plot
G = nx.Graph()

# Create connections between nodes
for k, v in d[0].items():
    G.add_edge(k[0], k[1], weight=(v * 10))
    
G.add_node("test", weight=100)

fig, ax = plt.subplots(figsize=(13,16))

pos = nx.spring_layout(G, k=1)

for key, value in pos.items():
