Next we'll create a function that'll removes Urls, lower case all alphabets and remove all stopwards called text_process.

In [5]:
def text_process(txt):
    """"
    Removes the following: 
            URLs
            Punctuations
            Stop words
    Return list of clean text words
    
    Parameters
    -------------------------------------------------------------------
    txt: string 
        A text string that you want to parse 
        
    Returns
    -------------------------------------------------------------------
    Clean words 
        
    """
    nonpunc = " ".join(re.sub("([^0-9A-Za-z \t])|(\w+:\/\/\S+)", "", txt).split())
    #nonpunc = [char for char in txt if char not in string.punctuation]
    nonpunc = ''.join(nonpunc)
    return [word for word in nonpunc.split() if word.lower() not in stopwords.words('english')]

all_tweets_clean = [text_process(tweet) for tweet in all_tweets]

To calculate and plot word frequency, we'll use the python built-in library called collection. The collection.Counter object has a useful built-in method most_common that will return the most commonly used words and the number of times that they are used.

First we'll flatten the list using itertools. Then we create a counter using collection.Counter. Finally, most_common is used to pull the most common word used in our search.

In [7]:
flatten_tweets = list(itertools.chain(*all_tweets_clean))

count_flat_tweets = collections.Counter(flatten_tweets)

count_flat_tweets.most_common(15)
