Principal Component Analysis

John Manalastas | 26 Oct 2019

The technique is an unsupervised statistical method used to examine the interrelations among a set of variables in order to identify the underylying structure of those variables. This procedure is aka factor analysis.

PCA is usually used as a dimensionality reduction technique in the following areas: facial recognition, computer vision and image compression. It is also used to find patterns in data with high dimensionality; these areas includes finance, data mining, bioinformatics, and psychology.

Key words_

  • Dimensionality - number of random variables in a dataset or features
  • Correlations - shows how strongly two variables are related to each other. The value range from -1 to +1
  • Orthogonal - prepedicualr line with a highly correlated line. It usually represents a correlation
    of 0
  • Covariance Matrix - matrix consists of the covariances between the pairs of variables. The (i,j)th element is the covariance between i-th and j-th variable.
  • Components - linear transformation that chooses a variable system for the data set such that the greatest variance of the data set comes to lie on the first axies

We will work with a dataset that is within in sklearn called load_breast_cancer. The data is set to predict if a cancer is malignant or benign, though we are not interested in that.

Our PCA, will explain which features explains the most variance in the dataset. We'll transform the data in a dataframe using pandas.
