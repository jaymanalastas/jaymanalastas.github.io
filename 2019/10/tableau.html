Tableau

John Manalastas | Wed 23 October 2019

Tableau is a powerful tool used to prepare raw data into easily understood format.

The data analysis is very fast with the software and the visualizations created are in the form of dshaboards and worksheets.

Tableau's advantages:

  • Great visualization tool for analysis
  • Qick and easy representation of big data which helps in resolving big data issues
  • The software can be used without a coding background

Let's get started:¶

First we need to select a dataset. In our case we will explore a pre-populated dataset in Tableau called World Indicators.

The dataset has the following features:

  • Brithrate
  • Business Tax Rate
  • CO2 Emission
  • Country
  • Ease of Business
  • Energy Usage
  • GDP
  • etc...

Our first screen should give us the rows and columns of the dataset and should look like this:

_¶

What we really need to do first is open up a sheet. This is where we will manipulate the data.

_¶

Dimensions - are qualitative values, (such as names, dates, or geographical data) We can use this to add details in our data. This field affect the level of dtail in the view.

Measures - are numeric values that can be measured. We add it to our map to view the data in the chart.

Filters - the data put in here is similar to passing an index match in an excel sheet. We are able to pick which data we want to show easily. In our example, I chose to filter $Years$ and only selected 2012 data.

Marks - this section controls color, size, shape, detail, text, and tooltips for marks in the view. Those dots in our figure above can be manipulated via this section.

¶

_¶

Dashboard¶

This is where we put our sheet for display. The dashboard highlighted above shows us how our data will be displayed when we send this to stakeholders.

Size - This is useful when presenting our data. we're able to customize the size depending on where we will present the data for an actual fit.

Sheets - This is where we can pull the sheets we've created and drag and drop the data into the main dashboard.

Conclusion¶

Tableau is the business intelligence software that provides an easy way to use the applications for fast analytics and visualization. It is simple to use and it can be taught to anyone, regardless if they've had coding exp. or not.
